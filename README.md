# SpendSmart - Budget Tracker

A mobile-first budget tracking app built with React and Vite.

## Features

- Add unlimited categories for expenses
- Track expenses with amounts and dates
- View total expenses
- Data persists in local storage

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open your browser to the provided URL

## Usage

- Add categories first
- Then add expenses to those categories
- View your spending summary
